# 第一章: LLM基础概念

本章将介绍大型语言模型(LLM)的基础概念，包括其发展历程、工作原理以及核心技术组件。

## 主题概览

- LLM的定义与历史
- 变压器架构详解（详见《Transformer深度解析》章节）
- 预训练与微调过程
- 模型评估指标
- 当前主流LLM对比

## 学习目标

完成本章后，你应该能够：
1. 理解LLM的核心概念和术语
2. 区分不同类型的LLM模型
3. 从工程角度理解Transformer架构的工作原理
4. 了解如何评估LLM的性能
